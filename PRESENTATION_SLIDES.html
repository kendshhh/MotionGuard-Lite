Slide 2

Project Purpose

  • Build a functional Python terminal system for guided self-defense practice.
  • Combine terminal interaction, file handling, and optional OpenCV webcam validation.
  • Demonstrate course programming concepts through a practical real-world problem.
Slide 3

Main Features

Training ModeChoose a technique and generate an automatic session summary.
Technique GuideView Block, Punch, and Escape instructions.
Progress TrackingSave and review practice records in a text file.
Webcam ValidationUse OpenCV for live technique feedback and fullscreen viewing.

Data Typesint, float, str, bool
OperatorsArithmetic, relational, and logical operators
Control Structuresif, elif, else for menu and scoring logic
Loopsfor and while loops for user interaction and processing
FunctionsReusable functions for menu flow, scoring, validation, and display
List HandlingLists for techniques, steps, feedback, and progress records
Slide 5

File Handling and Error Handling

  • File Handling: creates, reads, and updates progress.txt.
  • Additional File Use: temporary JSON validation data for OpenCV fallback flow.
  • Error Handling: validates user input and prevents crashes on missing files or invalid values.
  • Exception Handling: uses try/except for webcam validation, file access, and interpreter fallback.
Slide 6

System Demonstration Flow

  1. Open the terminal app.
  2. Select Start Training.
  3. Choose Block, Punch, or Escape.
  4. Choose whether to use OpenCV webcam validation.
  5. Review live feedback and finish the session.
  6. Show saved progress and computed results.
Slide 7

How Errors Are Handled

  • Invalid menu choices are rejected safely.
  • Invalid login or registration input is rejected before account actions continue.
  • Back and cancel options allow safe return to the main menu.
  • Missing OpenCV in the active interpreter triggers a retry using the project virtual environment.
  • Camera and pose-detection problems display readable feedback instead of crashing the program.
Slide 8

Conclusion

MotionGuard Lite is a complete Python terminal project that satisfies the required programming concepts, demonstrates practical problem solving, and provides a functional user-facing system with robust handling of both user input and system-level errors.
